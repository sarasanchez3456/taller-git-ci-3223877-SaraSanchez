# Proyecto de Ejemplo

Este es un proyecto de ejemplo con estructura básica.

## Estructura

- `src/`: Código fuente
  - `index.html`: Página principal
  - `app.js`: Lógica de la aplicación
- `tests/`: Pruebas
  - `suma.test.js`: Pruebas para la función suma

## Instalación

```bash
npm install
```

## Ejecutar pruebas

```bash
npm test
```

## Ejecutar aplicación

Abrir `src/index.html` en el navegador.
